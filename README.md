# Boncyber Cafe — Site statique

Site simple pour Boncyber Cafe (impression & services). Contenu : HTML + CSS. Prêt à publier gratuitement sur GitHub Pages, Netlify ou Vercel.

Comment publier sur GitHub Pages :
1. Ajoute ces fichiers (`index.html`, `styles.css`, `README.md`) à la racine du dépôt `mohamedalyamadouba-rgb/Boncyber`.
2. Commit & push sur la branche `main`.
3. Dans les settings du dépôt → Pages → Source : choisis `main` branch / root. Sauvegarde.
4. L'URL ressemblera à `https://<username>.github.io/Boncyber`.

Remarques :
- Remplace `your-form-id` dans le formulaire par ton identifiant Formspree si tu veux recevoir des messages.
- Pour un logo, ajoute une image dans `assets/logo.png` et remplace le texte du header par `<img src="assets/logo.png" alt="Boncyber Cafe">`.
- Si tu veux des couleurs particulières, donne‑moi une couleur au format hex (ex. `#ff6600`) et je mettrai à jour `styles.css`.

Si tu veux que je publie directement :
- J'ai poussé ces fichiers dans le dépôt `mohamedalyamadouba-rgb/Boncyber` sur la branche `main`.
- Pour activer GitHub Pages, va dans Settings → Pages et sélectionne "main" branch / root, puis sauvegarde. L'URL apparaîtra ensuite.
